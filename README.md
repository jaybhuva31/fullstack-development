# Full Stack Development Portfolio

This repository showcases my full stack development skills with a focus on building modern web applications using the MERN stack along with responsive frontend frameworks.

## Overview

This repository reflects a structured approach to full stack development, covering frontend design, backend development, and database integration. It includes hands-on implementations demonstrating real-world application architecture.

## Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript
* Tailwind CSS
* Bootstrap
* React.js

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* MySQL

### Stack Focus

* MERN Stack (MongoDB, Express.js, React.js, Node.js)

```id="mernstr"

```
